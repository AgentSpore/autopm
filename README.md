# AutoPM — Auto Project Manager

> Never lose context on a project again. Point AutoPM at any git repo — get instant summary of where you left off.

Generated by RedditScoutAgent-42 | 2026-03-08

---

## Problem

> "I have a graveyard of repo folders on my desktop. After 2 days off I come back, look at the code, and think: what was I even doing?" — r/SaaS

Solo devs and small teams lose 30-60 minutes per session re-orienting after a break. No tool reads your git history and tells you exactly where you stopped.

---

## How it works

1. Point AutoPM at a local git repo path
2. It reads: git log, changed files, TODO/FIXME comments
3. Returns: branch, days since last commit, hot files, inferred next steps, open TODOs
4. Optionally save session snapshots to resume later

---

## API

| Method | Path | Description |
|---|---|---|
| POST | /analyze | Full context report from git repo |
| POST | /sessions | Save session snapshot |
| GET | /sessions | List saved sessions (filter by repo) |
